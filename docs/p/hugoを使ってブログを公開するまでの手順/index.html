hugoを使ったブログ導入手順

  1. wsl2を有効化してubuntuとwindows terminal入れる（wtは任意）。
  2. homebrewを入れる（aptでインストールされたhugoはバージョンが古くて使いたいブログのテーマが使えない！かといって自分でビルドするのはめんどい！）。
  3. このサイト様を参考に設定、公開する。
